# Iglesia del Dios Viviente

Este es el sitio web oficial de la **Iglesia del Dios Viviente**, desarrollado para compartir información sobre ministerios, eventos y contacto.

## Contenido del proyecto
- `index.html`: Página principal
- `eventos.html`: Calendario de actividades
- `ministerios.html`: Información de ministerios
- `contacto.html`: Formulario de contacto
- `style.css`: Estilos del sitio
- `quienes-sosmo`: nuestra historia, misión y visión
